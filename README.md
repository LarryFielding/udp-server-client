# upd-server-client
Servidor y cliente sockets tipo datagrama

Para compilar:

```make```

Para ejecutar el servidor:

```./main_s```

Para ejecutar el cliente:

```./main_c```

En el caso del cliente especificar el puerto `7200`, que es el puerto fijo del servidor y posteriormente la direcci√≥n ip (la debe obtener directamente del servidor).
